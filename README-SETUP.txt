Nick & Libby Wedding Website - GoDaddy Upload Notes
===================================================

Files included:
- index.html: homepage with event information, registry link, and photo timeline
- directions.html: venue address, embedded map, map links, and calendar file
- rsvp.html: RSVP form with guest meal choices
- thank-you.html: RSVP confirmation page
- styles.css and script.js: design and RSVP guest-form functionality
- assets/images: optimized copies of the uploaded photos, arranged in the order provided
- nick-libby-wedding.ics: calendar invite file

How to upload to GoDaddy:
1. Open your GoDaddy hosting file manager or FTP client.
2. Upload the contents of this folder into your public website folder, usually public_html or the root folder for nickandlibby.com.
3. Make sure index.html is in the root folder, not inside another nested folder.
4. Visit https://nickandlibby.com to test the homepage.

Important RSVP note:
The RSVP form is set to submit through FormSubmit to:
nickandlibbywedding26@gmail.com

Because this is a static website-friendly form solution, the first form submission may trigger a confirmation email that must be approved from the wedding email inbox before future RSVPs are delivered. After uploading the site, submit one test RSVP and check the wedding email inbox/spam folder.

If you later use a GoDaddy plan with server-side code or a form builder, the RSVP form can be swapped to that service while keeping the same page design.
